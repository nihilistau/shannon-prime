qaic-generated FastRPC stub files for the sp_hex IDL interface.

These files are generated by running:
    build.cmd arm
in the scaffold directory (requires Hexagon SDK 5.5.6.0 with qaic on PATH).

The generated files to copy here after building:
    sp_hex_stub.c   — ARM-side FastRPC stub (compiled into libsp_hex.so OR
                      directly into sp-engine when SP_ENGINE_HEXAGON_FASTRPC=ON)
    sp_hex.h        — qaic-generated header (included by shannon_prime_hexagon.c)

These are committed here so the engine Android cross-build can use them
without requiring the full scaffold build environment on the CI host.

Re-generate whenever sp_hex.idl changes:
    cd scaffold && build.cmd arm
    copy build\android_aarch64\ship\sp_hex_stub.c src_generated\
    copy build\android_aarch64\ship\sp_hex.h      src_generated\
